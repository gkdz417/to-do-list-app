# To-Do List App

Basit bir görev takip uygulaması. Görev ekleyebilir, öncelik atayabilir, tamamlananları işaretleyebilir ve silebilirsiniz. Veriler tarayıcıda saklanır, sayfa kapansa bile silinmez.

## Canlı Demo

https://gkdz417.github.io/to-do-list-app/

## Teknolojiler

- HTML, CSS, JavaScript
- LocalStorage

## Özellikler

- Görev ekleme, silme, tamamlama
- Öncelik seviyeleri (Yüksek / Normal / Düşük)
- Filtreleme (Tümü / Aktif / Tamamlanan)
- LocalStorage ile kalıcı kayıt
